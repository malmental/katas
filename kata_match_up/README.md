# Kata Match UP

Juego de cartas de memoria con números romanos.

## Instalación

1. Clonar el repositorio:
```bash
git clone https://github.com/malmental/katas
cd kata_match_up
```

2. Ejecutar con PHP:
```bash
php -S localhost:8000
```

3. Abrir en el navegador:
```bash
http://localhost:8000/index.php
```
